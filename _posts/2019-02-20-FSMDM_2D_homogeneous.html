--- title: Simulation in a 2D homogeneous medium using the frequency-specific mixed domain method ---


This example shows how to simulate the wave propagation in a 2D homogeneous medium with the frequency-specific mixed domain method and the focused beam is generated with the phased-array.

Generate the grid structure in the computational domain


This simulation setup is based on the Generate the 2D focused ultrasound beam with a phased array using the transient mixed domain method. The phase delay for each element in the transducer are the same in the two cases. We first need to define the temporal and spatial computational domain in 2D forward simulations. In FSMDM, time step and temporal domain size are set to 0. 

         
dx = 3.75e-04;   %  step size in the x direction [m] 
dy = 3.75e-04;   %  step size in the y direction [m] 

x_length = 0.0600;   % total computatoinal domain size in x direction [m] 
y_length = 0.0330;   % total computatoinal domain size in y direction [m] 
mgrid = set_grid(0, 0, dx, x_length, dy, y_length);

Excitation signal

Continuous sine wave is used as the excitation signal. 

  
p0 = 1.0e6;    % pressure magnitude at the source [Pa]   
source_p = p0*exp(1i*omega_c*delay);  % define the pulse   
source_p = source_p.*TR_mask.';  % apply the transducer mask

Define the homogeneous medium. 


medium.c    = 1500;   % speed of sound [m/s]      
medium.rho  = 1000;   % density [kg/m^3]             
medium.beta = 3.6;    % nonlinear coefficient     
medium.ca   = 0;      % attenuation coefficient [dB/(MHz^y cm)]     
medium.cb   = 2.0;    % power law exponent

2D forward simulation

Fundamental pressure field is calculated with the 2D forward simulation function Forward2D_fund. Pressure at the second-harmonic frequency is calculated with the function Forward2D_sec. 


% forward propagation of the fundamental pressure   
p_fundamental = Forward2D_fund(mgrid, medium, source_p); 
% forward propagation of the second-harmonic pressure   
p_second = Forward2D_sec(mgrid, medium, p_fundamental, omega_c);
