--- title: Simulation in the 3D homogeneous medium using the frequency-specific mixed domain method ---


This example shows how to simulate the wave propagation in the 3D homogeneous medium with the frequency-specific mixed domain method and the focused beam is generated with the phased-array.

Generate the grid structure in the computational domain


This simulation setup is based on the Generate the 3D focused ultrasound beam with a phased array. The phase delay for each element in the transducer are the same in the two cases. We first need to define the temporal and spatial computational domain in 3D forward simulations. In FSMDM, we can set time step and temporal domain size to 0.

dx = 3.75e-04; % step size in the x direction [m]
dy = 3.75e-04; % step size in the y direction [m]
dz = 3.75e-04; % step size in the z direction [m]
x_length = 0.0454; % total computatoinal domain size in x direction [m]
y_length = 0.0454; % total computatoinal domain size in y direction [m]
z_length = 0.0330; % total computatoinal domain size in z direction [m]
mgrid = set_grid(0, 0, dx, x_length, dy, y_length, dz, z_length);

Excitation signal

Continuous sine wave is used as the excitation signal.

p0 = 1.0e6; % pressure magnitude at the source [Pa]
source_p = p0*exp(1i*omega_c*delay); % define the pulse
RHO = sqrt(X.^2+Y.^2);
source_p(RHO>TR_radius) = 0; % filter out the signals outside of the transducer

Define the homogeneous medium
Define the homogeneous medium.

medium.c = 1500; % speed of sound [m/s]
medium.rho = 1000; % density [kg/m^3]
medium.beta = 3.6; % nonlinear coefficient
medium.ca = 0; % attenuation coefficient [dB/(MHz^y cm)]
medium.cb = 2.0; % power law exponent

3D forward simulation

Fundamental pressure field results are calculated with the 3D forward simulation function Forward3D_fund. and the pressure at the second-harmonic frequency is calculated with the function Forward3D_sec.

p_fundamental = Forward3D_fund(mgrid, medium, source_p);
p_second = Forward3D_fund(mgrid, medium, p_fundamental, omega_c);
