--- title: Simulation in the 2D heterogeneous media with strong speed of sound contrast using the transient mixed domain method ---


This example provides the demenstration of using the transient mixed domain method for simulating wave propagation in 2D heterogeneous media with strong speed of sound contrast. TMDM is accurate for wave propagation in media with weakly speed of sound heterogeneity. To extend the MDM for modeling wave propagation in strongly heterogeneous media (e.g., skull), phase and amplitude corrections to the MDM are proposed. This example shows how to apply the corrections for simulations. Generally, for simulations with soft tissue or strongly heterogeneous media with small speed of sound contrast (contrast ratio less than 1.10), it is unnecessary to apply corrections.

Generating the grid structure to define the computational domain


We first need to define the temporal and spatial computational domain in 2D forward simulations with the funciton set_grid. For 2D problems, six inputs are required to to call this function. Coordinates defined within the mgrid structure are all in the Cartesian coordinates. 

        
dx = 5.3571e-04;    % step size in the x direction [m] 
dy = 1.3393e-04;    % step size in the y direction [m] 
dt = 1.1905e-07;    % time step size [s] 

x_length = 0.1671;      % total computational domain size in x direction [m] 
y_length = 0.0643;      % total computational domain size in y direction [m] 
t_length = 6.8571e-05;  % total temporal domain size [s] 
mgrid = set_grid(dt, t_length, dx, x_length, dy, y_length);

2D forward simulation

Transient pressure fields are calculated with the 2D forward simulation function. The figure below shows the (a) time-domain signal and (b) frequency-domain signals recorded at the transducer focus. Forward2D. 


% define where the pressure will be recorded

sensor_mask(100:300, :) = 1;
