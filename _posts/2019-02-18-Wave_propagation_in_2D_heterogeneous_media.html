--- title: Wave propagation in 2D heterogeneous meida ---

Generate the grid structure in the computational domain

To define the temporal and spatial computational domain in 2D forward simulations, we start from the funciton set_grid. For 2D problems, six inputs are required to to call this function. Coordinates defined with the mgrid functiions are all in Cartesian coordinates.

dx = 3.75e-04; % step size in the x direction [m]
dy = 3.75e-04; % step size in the y direction [m]
dt = 6.25e-08; % time step size [s]
xlen = 0.0735; % total computatoinal domain size in x direction [m]
ylen = 0.0210; % total computatoinal domain size in y direction [m]
tlen = 4.40e-05; % total temporal domain size [s]
mgrid = set_grid(dt, tlen, dx, xlen, dy, ylen);

Define the phased array transducer

In this example, phased array transducer is used to generate the foucsed ultrasound beam. Transducer parameters are shown in the code below. Details of defining a phased array transducer can refer to phased array.

p0 = 1; % transducer pressure magnitude [Pa]
fc = 1.0e6; % center frequency [Hz]
TR_focus = 0.0210; % transducer focus length [m]
TR_radius = 0.0135; % Transducer diameter [m]

Define the excitation signal

A four-cycles gaussian-modulated pulse is used for the excitation signal and can read Generate the 2D focused ultrasound beam with a phased array for details.

Define the media

Speed of sound, density, nonlinear coefficient, attenuation coefficient and power law exponent can all be spatially varying functions.

medium.c = c; % speed of sound [m/s]
medium.rho = rho; % density [kg/m^3]
medium.beta = beta; % nonlinear coefficient
medium.ca = ca; % attenuation coefficient [dB/(MHz^y cm)]
medium.cb = cb; % power law exponent

2D forward simulation

Transient pressure field results are calculated with the 2D forward simulation function Forward2D.

p_total = Forward2D(mgrid, medium, excit_p);
