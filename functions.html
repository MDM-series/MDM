Functions

set_grid

set_grid is the grid class used across the mSOUND Toolbox and it can generate a structure contains the Cartesian grid coordinates, evenly spaced time array and wavenumber matrices used within the simulation in mSOUND. 

 
mgrid = set_grid(dt, t_length, dx, x_length);                         
mgrid = set_grid(dt, t_length, dx, x_length, dy, y_length);     
mgrid = set_grid(dt, t_length, dx, x_length, dy, y_length, dz, z_length);
